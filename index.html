Countdown Timer
Simple Javascript Countdown Timer
Created by Edward Aubrey

Enter a date for the countdown to begin!


Insert the following into your <body>

<script type="text/javascript" src="countdown.js"></script>
<script>
  //Time to countdown (YYYY, MM, DD, HH, MM, SS):
  //NOTE (Month counts from 0, so enter the month -1 eg. Jan: 0, Feb: 1...)
  endTime = new Date(2017, 10, 7, 20, 00);
</script>

Single object (HH:MM:SS)


Counting down to:

<p class="clock" id="clock"></p>

Multiple objects (HH, MM, SS)

hours, minutes, seconds.

Counting down to:

<p>
    <span class="clock" id="hours"></span>
    <span class="clock" id="minutes"></span>
    <span class="clock" id="seconds"></span>
</p>
